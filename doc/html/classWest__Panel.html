Pulsante aggiungi: apre un filechooser per la scelta del file da inserire, successivamente apre un form per l'aggiunta dei campi specifici del file;

Pulsante modifica: apre lo stesso form di aggiungi ma consente la modifica dei valori di una riga selezionata;

Pulsante informazioni: apre una piccola finestra contenente le informazioni specifiche e non di un singolo file selezionato;

Pulsante riproduci: apre un piccolo player che riproduce singole traccie selezionate dall'utente;
